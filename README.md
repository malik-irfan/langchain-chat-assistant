# langchain-chat-assistant
A Streamlit app integrating LangChain chains with Groq LLM models for powerful and scalable AI-driven solutions.

## Overview

This application leverages LangChain’s flexible chain management and Groq’s cutting-edge machine learning models to build an AI-powered tool that can be easily deployed and customized. The app is built with Streamlit for an intuitive and interactive web interface.

## Features

- **LangChain Integration**: Seamlessly manage and execute LLM chains for complex NLP tasks.

- **Groq LLM Models**: Use Groq’s optimized models to perform high-speed, scalable language tasks.

- **Interactive UI**: Built using Streamlit, providing an easy-to-use interface for users to interact with the AI.

## Installation

1. Clone the repository:

`git clone https://github.com/malik-irfan/langchain-chat-assistant.git
cd langchain-groq-streamlit`


2. Install dependencies:

`uv add -r requirements.txt`


3. Run the app:

`streamlit run app.py`


4. Visit http://localhost:8501 in your browser to see the app in action.

## Technologies Used

- **LangChain**: A framework for building language model chains.

- **Groq**: High-performance LLMs for fast and scalable AI tasks.

- **Streamlit**: A framework for creating interactive web apps with minimal code.
